Containerize Django Application

Prerequisites

First install Python and Pip on your Operating System or EC2 instance or any VM.

Install Django using pip/python; Globally or Virtual Environment(recommended).

To verify installation, run {$django-admin --version} command.

Create your project by running the command: {$django-admin startproject [project-name]}.

Create your application by running the command: {$python3 manage.py startapp [application-name]}.

Create a templetes folder in your application and store your HTML or CSS file. (application/template/file.html)

Create a simple python function to render your HTML file.

Containerize Django Application

First create a Docker File with set of instructions to build a Docker Image.

Submit your Docker File to Docker Daemon using 'Build' command to create Image.

Your Image is ready and to create your Docker Container, use 'Run' command.

This docker container is your final output, which contains your application, application libraries, and system dependencies.

Share your Docker Container Image to Docker Hub using 'Push' command.

Alright your application is now public.
